## **1. Fundamentals of Web Development (Weeks 1–3)**

### 🔹 Learn:

- **HTML**: Structure of web pages
    
- **CSS**: Styling elements
    
- **JavaScript**: Making web pages interactive
    

### ✅ Topics:

- HTML tags: headings, paragraphs, images, links, forms
    
- CSS basics: selectors, box model, flexbox, media queries
    
- JavaScript: variables, data types, functions, events, DOM
    

### 📘 Resources:

- [HTML & CSS](https://developer.mozilla.org/en-US/docs/Learn/HTML)
    
- [JavaScript](https://javascript.info/)
    
- FreeCodeCamp – Responsive Web Design
    

### 🧠 Exercises:

- Create a personal portfolio page (HTML & CSS)
    
- Build a basic to-do list (JavaScript)
    
- Style a form using Flexbox
    

---

## **2. Front-End Development (Weeks 4–8)**

### 🔹 Learn:

- Front-end frameworks (choose one to start; I recommend **React**)
    
- Component-based architecture
    
- State management, props, routing
    

### ✅ Topics:

- JSX syntax, functional components
    
- React hooks (`useState`, `useEffect`)
    
- React Router for navigation
    

### 📘 Resources:

- React Docs
    
- Codecademy – Learn React
    
- [Frontend Mentor](https://www.frontendmentor.io/)
    

### 🧠 Projects:

- Weather App using an API
    
- Blog UI (Home, About, Posts page)
    
- Recipe Search App with routing
    

---

## **3. Back-End Development (Weeks 9–13)**

### 🔹 Learn:

- Server-side language: **Node.js** with Express (or Python with Flask/Django)
    
- Databases: **SQL** (MySQL/PostgreSQL) + **NoSQL** (MongoDB)
    
- RESTful APIs
    

### ✅ Topics:

- Creating servers and routes
    
- CRUD operations
    
- Middleware and authentication basics
    

### 📘 Resources:

- [Node.js & Express Docs](https://expressjs.com/)
    
- [MongoDB University](https://university.mongodb.com/)
    
- [PostgreSQL Tutorial](https://www.postgresqltutorial.com/)
    

### 🧠 Projects:

- Simple REST API for notes
    
- User Authentication System (JWT + hashed passwords)
    
- Expense Tracker with MongoDB
    

---

## **4. Full Stack Integration (Weeks 14–18)**

### 🔹 Learn:

- Connect front-end and back-end using **APIs**
    
- Use **axios** or **fetch** for data handling
    
- Manage authentication, session/token storage
    

### ✅ Topics:

- CORS, cookies, and token-based auth
    
- Frontend state sync with backend
    
- MVC pattern
    

### 📘 Resources:

- Full Stack Open
    
- MERN Stack Tutorial
    

### 🧠 Final Project:

**Full Stack Blog App**

- Frontend in React
    
- Backend with Node.js + MongoDB
    
- Features: login/signup, create/edit/delete posts, comments

### 🔹 Learn:

- Hosting: **Vercel**, **Netlify**, **Render**, **Railway**, or **Heroku**

- Deployment with Vercel
    
- [Best practices](https://12factor.net/)
    

### 🧠 Tasks:

- Host your blog app
    
- Practice version control with branches and PRs
    
- Add error boundaries and input sanitization
