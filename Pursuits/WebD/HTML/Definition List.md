In HTML, a definition list (`<dl>`) is used to define terms and their corresponding descriptions. It consists of the following elements:

- `<dl>`: This is the container for the definition list.
- `<dt>`: Stands for "definition term". It is used to define the term being described.
- `<dd>`: Stands for "definition description". It provides the description or definition of the term defined by `<dt>`.
