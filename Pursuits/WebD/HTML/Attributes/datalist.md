**Role**: The `<datalist>` tag provides a list of predefined options for an `<input>` element. It allows the user to select from a list of suggestions, but it also permits the user to type their own value if the desired option is not available in the list.
```html
<form>
        <input type="text" name="" list="courses"></intput>
        <datalist id="courses">
            <option value="Python">Python</option>
            <option value="DBMS">DBMS</option>
            <option value="Stack">Full Stack</option>
        </datalist>
    </form>
```
