**Role**: The `<select>` tag is used to create a dropdown list, allowing the user to choose one or more options from a predefined list of choices.
```html
<form>
        <select>
            <option value="CSE">BE CSE</option>
            <option value="ME">BE ME</option>
            <option value="EC">BE EC</option>
        </select>
    </form>
```
