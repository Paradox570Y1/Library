# User Experience
- `Simplicity
	Keeping things simple is always better than making it more complex or more cluttered.
- `Consistency`
	Keep your designs consistent but also keep the functionality consistent. Like make sure your nav bar remains consistent in multi page website might add some functionality for specific page but make sure positioning of basic elements like search remains same.
- `Reading Patterns`
	Take into account how human eyes read the webpage and which parts they look at while browsing
	Most common reading patter is F pattern
	![[Pasted image 20240805094728.png|300]]
	![[Pasted image 20240805094838.png|300]]
	Then comes Z-pattern
	![[Pasted image 20240805094935.png|300]]
	This might be for website with video content.
	Popularly used by facebook.
	![[Pasted image 20240805095043.png|300]]

- Sites designed for landscape mode does not look good on phone.
![[Pasted image 20240805095203.png|300]]
- Sites designed for mobiles when rendered on laptops have so much blank space.
![[Pasted image 20240805095355.png|400]]

- Try to avoid so many banners on mobile version.
![[Pasted image 20240805095502.png|200]]

- Don't make pictures so big that user has to keep scrolling
