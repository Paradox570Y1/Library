- 1em means if parent element is 20px then the child element will also be 20px. Similarly 2em means twice of 20px. Its size is relative to parent element.
- 1 rem means size will be 1x of size written in html element. \<html> element is the root element.
- In general font-size of root element html is 16 px , so initially 1em and 1rem is 16px.
