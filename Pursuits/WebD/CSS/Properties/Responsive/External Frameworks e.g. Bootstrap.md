#### Pro's
- Bootstrap are useful if want to develop our website quickly and easily.
- Consistent styling across the website.
- It's like you hired a professional developer.
- Really good browser compatibuility.

#### Con's
- Using external css framework , we are adding everything into classes which results in class bloat.
![[Pasted image 20240730100431.png|200]]
- Less clean to read and difficult to modify.

## When to use Bootstrap
- When you are building a mobile first responsive website quickly and access components designed by professional designers.

## When not to use Bootstrap
- If you are building simple website with html and css  or you are building a really complex design website and you want complete control over.
