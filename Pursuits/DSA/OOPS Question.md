Problem Statement 1
You are required to implement a booking system for a travel agency. Create a base class booking with a method calculatePrice(). Create two derived classes FlightBooking and HotelBooking that override the calculatePrice() method to provide specific implementations. Additionally, overload the calculatePrice() method in booking to handle different types of discounts.

Input: -
•	Creating of Booking, FlightBooking, and HotelBooking objects with their respective attributes.
Output: -
•	Calculation of booking price with or without discounts using overloaded and overridden methods.




Problem Statement 2
You are required to implement a basic university management system. The student and Professor classes should inherit from the Person class. The Person class will have attributes name and age, and methods to get these attributes. The Student class will have an additional attribute studentId and a method to get this attribute. The Professor class will have an additional attribute professorId and a method to get this attribute.

Input: -
•	Creation of Person, Student, and Professor objects with their respective attributes.
Output: -
•	Display of the name, age, studentId, and professorId for the respective objects using the methods from all three classes.



Problem Statement 3
You are required to implement a basic shape management system using abstract classes and methods in Java. Create an abstract base class shape with an abstract method calculateArea(). Create two derived classes Circle and Rectangle that implement the calculateArea() method to provide specific calculations for their respective shapes.

Input: -
•	Creation of Shape, Circle, and Rectangle objects with their respective attributes.
Output: -
•	Calculations of the area for the respective shapes using the implemented calculateArea() method.
