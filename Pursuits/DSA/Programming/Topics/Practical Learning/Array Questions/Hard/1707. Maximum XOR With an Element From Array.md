[1707. Maximum XOR With an Element From Array](https://leetcode.com/problems/maximum-xor-with-an-element-from-array/)

You are given an array `nums` consisting of non-negative integers. You are also given a `queries` array, where `queries[i] = [xi, mi]`.

The answer to the `ith` query is the maximum bitwise `XOR` value of `xi` and any element of `nums` that does not exceed `mi`. In other words, the answer is `max(nums[j] XOR xi)` for all `j` such that `nums[j] <= mi`. If all elements in `nums` are larger than `mi`, then the answer is `-1`.

Return _an integer array_ `answer` _where_ `answer.length == queries.length` _and_ `answer[i]` _is the answer to the_ `ith` _query._

**Example 1:**

**Input:** nums = [0,1,2,3,4], queries = [[3,1],[1,3],[5,6]]
**Output:** [3,3,7]
**Explanation:**
1) 0 and 1 are the only two integers not greater than 1. 0 XOR 3 = 3 and 1 XOR 3 = 2. The larger of the two is 3.
2) 1 XOR 2 = 3.
3) 5 XOR 2 = 7.

**Example 2:**

**Input:** nums = [5,2,4,6,6,3], queries = [[12,4],[8,1],[6,3]]
**Output:** [15,-1,5]

**Constraints:**

- `1 <= nums.length, queries.length <= 105`
- `queries[i].length == 2`
- `0 <= nums[j], xi, mi <= 109`



# Using Trie with bit manipulation

```java
class Solution {
    class Node{
        Node[] bit;
        int min;
        Node() {
            bit = new Node[2];
            min = (int) 1e9 + 1;
        }

        public Node get(int b) {
            return bit[b];
        }

        public void put(int b) {
            bit[b] = new Node();
        }

        public void updateMin(int limit) {
            min = Math.min(min, limit);
        }

        public boolean contains(int b) {
            return bit[b] != null;
        }

        public boolean belowLimit(int num) {
            return min <= num;
        }
    }

    class Trie {
        Node root;
        Trie() {
            root = new Node();
        }

        public void insert(int num) {
            Node trav = root;
            for (int i = 31; i >= 0; i--) {
                int bit = (num >> i) & 1;
                if (!trav.contains(bit)) trav.put(bit);
                trav.updateMin(num);
                trav = trav.get(bit);
            }
            trav.updateMin(num);
        }

        public int getMax(int[] query) {
            int num = query[0], limit = query[1];
            Node trav = root;
            if (!trav.belowLimit(limit)) return -1;
            int res = 0;
            for (int i = 31; i >= 0; i--) {
                int bit = (num >> i) & 1;
                if (trav.contains(1-bit) && trav.get(1-bit).belowLimit(limit)) {
                    res = res | (1 << i);
                    trav = trav.get(1-bit);
                } else trav = trav.get(bit);
            }
            return res;
        }
    }
    public int[] maximizeXor(int[] nums, int[][] queries) {
        Trie trie = new Trie();
        for (int num : nums) {
            trie.insert(num);
        }
        int[] res = new int[queries.length];
        int i = 0;
        for (int[] query : queries) {
            res[i++] = trie.getMax(query);
        }
        return res;
    }
}
```

```java
class Solution {
    class Node{
        Node[] bit;
        Node() {
            bit = new Node[2];
        }

        public Node get(int b) {
            return bit[b];
        }

        public void put(int b) {
            bit[b] = new Node();
        }

        public boolean contains(int b) {
            return bit[b] != null;
        }
    }

    class Trie {
        Node root;
        Trie() {
            root = new Node();
        }

        public void insert(int num) {
            Node trav = root;
            for (int i = 31; i >= 0; i--) {
                int bit = (num >> i) & 1;
                if (!trav.contains(bit)) trav.put(bit);
                trav = trav.get(bit);
            }
        }

        public int getMax(int num) {
            Node trav = root;
            int res = 0;
            for (int i = 31; i >= 0; i--) {
                int bit = (num >> i) & 1;
                if (trav.contains(1-bit)) {
                    res = res | (1 << i);
                    trav = trav.get(1-bit);
                } else trav = trav.get(bit);
            }
            return res;
        }
    }
    class Query implements Comparable<Query> {
        int num;
        int limit;
        int idx;
        Query(int num, int limit, int idx) {
            this.num = num;
            this.limit = limit;
            this.idx = idx;
        }

        @Override
        public int compareTo(Query other) {
            return Integer.compare(this.limit, other.limit);
        }
    }
    public int[] maximizeXor(int[] nums, int[][] queries) {
        Trie trie = new Trie();
        Arrays.sort(nums);
        PriorityQueue<Query> pq = new PriorityQueue<>();
        for (int i = 0; i < queries.length; i++) {
            pq.offer(new Query(queries[i][0], queries[i][1], i));
        }
        int[] res = new int[queries.length];
        int i = 0;
        while (!pq.isEmpty()) {
            Query query = pq.poll();
            while (i < nums.length && nums[i] <= query.limit) {
                trie.insert(nums[i++]);
            }

            if (i == 0) res[query.idx] = -1;
            else res[query.idx] = trie.getMax(query.num);
        }
        return res;
    }
}
```
