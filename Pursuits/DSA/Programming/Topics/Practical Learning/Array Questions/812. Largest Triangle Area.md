[812. Largest Triangle Area](https://leetcode.com/problems/largest-triangle-area/)

Given an array of points on the **X-Y** plane `points` where `points[i] = [xi, yi]`, return _the area of the largest triangle that can be formed by any three different points_. Answers within `10-5` of the actual answer will be accepted.

**Example 1:**

![|300](https://s3-lc-upload.s3.amazonaws.com/uploads/2018/04/04/1027.png)

**Input:** points =` [[0,0],[0,1],[1,0],[0,2],[2,0]]`
**Output:** 2.00000
**Explanation:** The five points are shown in the above figure. The red triangle is the largest.

**Example 2:**

**Input:** points = `[[1,0],[0,0],[0,1]]`
**Output:** 0.50000

**Constraints:**

- `3 <= points.length <= 50`
- `-50 <= xi, yi <= 50`
- All the given points are **unique**.

# Using Math

```java
class Solution {
    public double largestTriangleArea(int[][] points) {
        int n = points.length;
        double res = 0.0;
        for (int i = 0; i+2 < n; i++) {
            for (int j = i+1; j+1 < n; j++) {
                for (int k = j+1; k < n; k++) {
                    double area = calArea(points[i], points[j], points[k]);
                    res = Math.max(res, area);
                }
            }
        }
        return res;
    }
    private double calArea(int[] p1,int[] p2, int p3[]) {
        int x1 = p1[0], x2 = p2[0], x3 = p3[0];
        int y1 = p1[1], y2 = p2[1], y3 = p3[1];
        return Math.abs(x1 * (y2 - y3) + x2 * (y3 - y1) + x3 * (y1 - y2))/2.0;
    }
}
```
