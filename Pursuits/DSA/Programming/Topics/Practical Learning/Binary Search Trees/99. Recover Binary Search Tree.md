[99. Recover Binary Search Tree](https://leetcode.com/problems/recover-binary-search-tree/)

You are given the `root` of a binary search tree (BST), where the values of **exactly** two nodes of the tree were swapped by mistake. _Recover the tree without changing its structure_.

**Example 1:**

![](https://assets.leetcode.com/uploads/2020/10/28/recover1.jpg)

**Input:** root = [1,3,null,null,2]
**Output:** [3,1,null,null,2]
**Explanation:** 3 cannot be a left child of 1 because 3 > 1. Swapping 1 and 3 makes the BST valid.

**Example 2:**

![](https://assets.leetcode.com/uploads/2020/10/28/recover2.jpg)

**Input:** root = [3,1,4,null,null,2]
**Output:** [2,1,4,null,null,3]
**Explanation:** 2 cannot be in the right subtree of 3 because 2 < 3. Swapping 2 and 3 makes the BST valid.

**Constraints:**

- The number of nodes in the tree is in the range `[2, 1000]`.
- `-231 <= Node.val <= 231 - 1`


# Optimal
- Using Inorder which is like sorted order to find the suspect nodes
```java
class Solution {
    TreeNode a=null,b=null,c=null;
    TreeNode cmp = new TreeNode(Integer.MIN_VALUE);
    private void swap(TreeNode a,TreeNode b){
        int temp = a.val;
        a.val = b.val;
        b.val = temp;
    }
    private void helper(TreeNode node){
        if(node==null)return;
        
        helper(node.left);
        int data=node.val;
        if(data<cmp.val){
            if(a==null){
                a=node;
                b=cmp;
            }
            else c=node;
        }
        else cmp =  node;
        helper(node.right);
    }
    public void recoverTree(TreeNode root) {
        helper(root);
        if(c==null)swap(a,b);
        else swap(b,c);
    }
}
```




# Wrong way

- This will not work because it only compares parent with left branch or right branch but their could be cases where nodes to be swapped are in left subtree and right subtree to the parent for which this logic doesn't account for.
```java
class Solution {
    public void recoverTree(TreeNode root) {
        helper(root, null, null);
    }
    private boolean helper(TreeNode root, TreeNode min, TreeNode max) {
        if (root ==  null) return false;
        if((min == null || root.val > min.val) && (max == null || max.val > root.val)) {
            if(helper(root.left, min, root) || helper(root.right, root, max)) return true;
            return false;
        }
        if(min == null) return swap(root, max);
        if(max == null || min.val > root.val) return swap(root, min);
        return swap(root, max);
    }
    private boolean swap(TreeNode n1, TreeNode n2) {
        int temp = n1.val;
        n1.val = n2.val;
        n2.val = temp;
        return true;
    }
}
```
