- In BST it matters what the left and right values are.
- Nodes in the left of current node are smaller than current.
- Nodes in the right of current node are greater than current.
- Above is true for subtrees as well so each subtree is also a BST
- Ideally, duplicates are not allowed in BST.
- But we can integrate duplicates in BST as well if needed.
	- We can either add them in the tree.
	- We can make pair of node and its frequency.
