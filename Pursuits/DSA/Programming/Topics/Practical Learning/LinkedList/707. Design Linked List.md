[707. Design Linked List](https://leetcode.com/problems/design-linked-list/)

Design your implementation of the linked list. You can choose to use a singly or doubly linked list.  
A node in a singly linked list should have two attributes: `val` and `next`. `val` is the value of the current node, and `next` is a pointer/reference to the next node.  
If you want to use the doubly linked list, you will need one more attribute `prev` to indicate the previous node in the linked list. Assume all nodes in the linked list are **0-indexed**.

Implement the `MyLinkedList` class:

- `MyLinkedList()` Initializes the `MyLinkedList` object.
- `int get(int index)` Get the value of the `indexth` node in the linked list. If the index is invalid, return `-1`.
- `void addAtHead(int val)` Add a node of value `val` before the first element of the linked list. After the insertion, the new node will be the first node of the linked list.
- `void addAtTail(int val)` Append a node of value `val` as the last element of the linked list.
- `void addAtIndex(int index, int val)` Add a node of value `val` before the `indexth` node in the linked list. If `index` equals the length of the linked list, the node will be appended to the end of the linked list. If `index` is greater than the length, the node **will not be inserted**.
- `void deleteAtIndex(int index)` Delete the `indexth` node in the linked list, if the index is valid.

**Example 1:**

**Input**
["MyLinkedList", "addAtHead", "addAtTail", "addAtIndex", "get", "deleteAtIndex", "get"]
[[], [1], [3], [1, 2], [1], [1], [1]]
**Output**
[null, null, null, null, 2, null, 3]

**Explanation**
MyLinkedList myLinkedList = new MyLinkedList();
myLinkedList.addAtHead(1);
myLinkedList.addAtTail(3);
myLinkedList.addAtIndex(1, 2);    // linked list becomes 1->2->3
myLinkedList.get(1);              // return 2
myLinkedList.deleteAtIndex(1);    // now the linked list is 1->3
myLinkedList.get(1);              // return 3

**Constraints:**

- `0 <= index, val <= 1000`
- Please do not use the built-in LinkedList library.
- At most `2000` calls will be made to `get`, `addAtHead`, `addAtTail`, `addAtIndex` and `deleteAtIndex`.


# Brute

```java
class MyLinkedList {
    class Node{
        int val;
        Node next, prev;
        Node(int val) {
            this.val = val;
            next = prev = null;
        }
    }
    private Node head, tail;
    private Map<Integer, Node> findNode; // index - Node
    private int size;
    public MyLinkedList() {
        this.head = new Node(-1);
        this.tail = new Node(-1);
        head.next = tail;
        tail.prev = head;
        this.findNode = new HashMap<>();
        this.size = 0;
    }
    
    public int get(int index) {
        if (findNode.containsKey(index)) return findNode.get(index).val;
        return -1;
    }
    
    public void addAtHead(int val) {
        addAtIndex(0, val);
    }
    
    public void addAtTail(int val) {
        addAtIndex(size, val);
    }
    private void shiftRight(int index, Node node) {
        for (int i = index; i < size; i++) {
            Node oldNode = findNode.get(i);
            findNode.put(i, node);
            node = oldNode;
        }
        findNode.put(size, node);
        size++;
    }
    public void addAtIndex(int index, int val) {
        if (index > size) return;
        Node node = new Node(val);
        Node prevNode = head;
        if (index > 0) {
            prevNode = findNode.get(index-1);
        }
        Node front = prevNode.next;
        prevNode.next = node;
        node.next = front;
        front.prev = node;
        node.prev = prevNode;
        shiftRight(index, node);
    }
    private void shiftLeft(int index) {
        for (int i = index; i < size-1; i++) {
            findNode.put(i, findNode.get(i+1));
        }
        findNode.remove(size-1);
        size--;
    }
    public void deleteAtIndex(int index) {
        if (index >= size) return;
        Node node = findNode.get(index);
        node.prev.next = node.next;
        node.next.prev = node.prev;
        shiftLeft(index);
        node.next = node.prev = null;
    }
}
```

```java
class MyLinkedList {
    private Map<Integer, Integer> findVal; // index - Node
    private int size;
    public MyLinkedList() {
        this.findVal = new HashMap<>();
        this.size = 0;
    }
    
    public int get(int index) {
        if (findVal.containsKey(index)) return findVal.get(index);
        return -1;
    }
    
    public void addAtHead(int val) {
        addAtIndex(0, val);
    }
    
    public void addAtTail(int val) {
        addAtIndex(size, val);
    }
    private void shiftRight(int index, int val) {
        for (int i = index; i < size; i++) {
            int oldVal = findVal.get(i);
            findVal.put(i, val);
            val = oldVal;
        }
        findVal.put(size, val);
        size++;
    }
    public void addAtIndex(int index, int val) {
        if (index > size) return;
        shiftRight(index, val);
    }
    private void shiftLeft(int index) {
        for (int i = index; i < size-1; i++) {
            findVal.put(i, findVal.get(i+1));
        }
        findVal.remove(size-1);
        size--;
    }
    public void deleteAtIndex(int index) {
        if (index >= size) return;
        shiftLeft(index);
    }
}
```
