[257. Binary Tree Paths](https://leetcode.com/problems/binary-tree-paths/)
Given the `root` of a binary tree, return _all root-to-leaf paths in **any order**_.

A **leaf** is a node with no children.

**Example 1:**

![](https://assets.leetcode.com/uploads/2021/03/12/paths-tree.jpg)

**Input:** root = [1,2,3,null,5]
**Output:** ["1->2->5","1->3"]

**Example 2:**

**Input:** root = [1]
**Output:** ["1"]

**Constraints:**

- The number of nodes in the tree is in the range `[1, 100]`.
- `-100 <= Node.val <= 100`

# Brute

```java
class Solution {
    public List<String> binaryTreePaths(TreeNode root) {
        List<String> ans = new ArrayList<>();
        dfs(root,ans,"");
        return ans;
    }
    private void dfs(TreeNode node,List<String> ans,String s){
        if(node == null){
            return;
        }
        if(node.left==null && node.right==null){
            ans.add(s+node.val);
            return;
        }
        dfs(node.left,ans,s+node.val+"->");
        dfs(node.right,ans,s+node.val+"->");
    }
}
```



# Using StringBuilder

```java
class Solution {
    public List<String> binaryTreePaths(TreeNode root) {
        List<String> ans = new ArrayList<>();
        dfs(root,ans,new StringBuilder());
        return ans;
    }
    private void dfs(TreeNode node,List<String> ans,StringBuilder s){
        if(node == null){
            return;
        }
        if(node.left==null && node.right==null){
            ans.add(s.toString()+node.val);
            return;
        }
        int n = s.length();
        s.append(node.val+"->");
        dfs(node.left,ans,s);
        s.setLength(n);
        n = s.length();
        s.append(node.val+"->");
        dfs(node.right,ans,s);
        s.setLength(n);
    }
}
```

# After Optimization

```java
class Solution {
    public List<String> binaryTreePaths(TreeNode root) {
        List<String> ans = new ArrayList<>();
        dfs(root,ans,new StringBuilder());
        return ans;
    }
    private void dfs(TreeNode node,List<String> ans,StringBuilder s){
        if(node == null){
            return;
        }
        int n = s.length();
        s.append(node.val);
        if(node.left==null && node.right==null){
            ans.add(s.toString());
        }
        else{
            s.append("->");
            dfs(node.left,ans,s);
            dfs(node.right,ans,s);
        }
        s.setLength(n);
    }
}
```
