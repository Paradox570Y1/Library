[1780. Check if Number is a Sum of Powers of Three](https://leetcode.com/problems/check-if-number-is-a-sum-of-powers-of-three/)

Given an integer `n`, return `true` _if it is possible to represent_ `n` _as the sum of distinct powers of three._ Otherwise, return `false`.

An integer `y` is a power of three if there exists an integer `x` such that `y == 3x`.

**Example 1:**

**Input:** n = 12
**Output:** true
**Explanation:** 12 = 31 + 32

**Example 2:**

**Input:** n = 91
**Output:** true
**Explanation:** 91 = 30 + 32 + 34

**Example 3:**

**Input:** n = 21
**Output:** false

**Constraints:**

- `1 <= n <= 107`

# Optimal

TC -> O(log N)
```java
class Solution {
    public boolean checkPowersOfThree(int n) {
        int flag=1;
        while(flag<=n){
            flag*=3;
        }
        flag/=3;
        int prev =-1;
        while(n>0){
            if(prev==flag)return false;
            n-=flag;
            prev=flag;
            while(flag>n){
                flag/=3;
            }
        }
        if(n==0)return true;
        return false;
    }
}
```

Using recursion
```java
class Solution {
    public boolean checkPowersOfThree(int n) {
        if(n==1) return true;
        if(n%3==0)return checkPowersOfThree(n/3);
        if((n-1)%3==0)return checkPowersOfThree((n-1)/3);
        return false;
    }
}
```

Using ternary presentation
```java
class Solution {
    public boolean checkPowersOfThree(int n) {
        while(n>0){
            int rem = n%3;
            if(rem==2)return false;
            if(n==0)return true;
            if(n==1)return true;
            if(n==2)return false;
            n/=3;
        }
        return false;
    }
}
```
