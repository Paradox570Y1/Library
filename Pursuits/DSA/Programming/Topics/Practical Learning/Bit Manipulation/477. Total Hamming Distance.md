[477. Total Hamming Distance](https://leetcode.com/problems/total-hamming-distance/)
The [Hamming distance](https://en.wikipedia.org/wiki/Hamming_distance) between two integers is the number of positions at which the corresponding bits are different.

Given an integer array `nums`, return _the sum of **Hamming distances** between all the pairs of the integers in_ `nums`.

**Example 1:**

**Input:** nums = [4,14,2]
**Output:** 6
**Explanation:** In binary representation, the 4 is 0100, 14 is 1110, and 2 is 0010 (just
showing the four bits relevant in this case).
The answer will be:
HammingDistance(4, 14) + HammingDistance(4, 2) + HammingDistance(14, 2) = 2 + 2 + 2 = 6.

**Example 2:**

**Input:** nums = [4,14,4]
**Output:** 4

**Constraints:**

- `1 <= nums.length <= 104`
- `0 <= nums[i] <= 109`
- The answer for the given input will fit in a **32-bit** integer.

# Brute

```java
class Solution {
    public int totalHammingDistance(int[] nums) {
        int res = 0;
        for (int i = 0; i < nums.length; i++) {
            for (int j = i+1; j < nums.length; j++) {
                if (nums[i] != nums[j]) {
                    int num = nums[i] ^ nums[j];
                    int count = 0; 
                    while (num != 0) {
                        count++;
                        num &= (num-1);
                    }
                    res += count;
                }
            }
        }
        return res;
    }
}
```


# Optimal

```java
class Solution {
    public int totalHammingDistance(int[] nums) {
        int[] bits = new int[32];
        for (int n : nums) {
            int mask = 1;
            for(int i = 0; i < 32; i++) {
                if ((n & mask) != 0) {
                    bits[i]++;
                }
                mask <<= 1;
            }
        }
        int res = 0;
        for (int bitCnt : bits) {
            res += bitCnt * (nums.length - bitCnt);
        }
        return res;
    }
}
```


```java
class Solution {
    public int totalHammingDistance(int[] nums) {
        int res = 0;
        for (int i = 0; i < 32; i++) {
            res += distance(nums, i);
        }
        return res;
    }
    private int distance(int[] nums, int pos) {
        int count = 0;
        for (int num : nums) {
            count += ((num >> pos) & 1);
        }
        return count * (nums.length - count);
    }
}
```
