[547. Number of Provinces](https://leetcode.com/problems/number-of-provinces/)

There are `n` cities. Some of them are connected, while some are not. If city `a` is connected directly with city `b`, and city `b` is connected directly with city `c`, then city `a` is connected indirectly with city `c`.

A **province** is a group of directly or indirectly connected cities and no other cities outside of the group.

You are given an `n x n` matrix `isConnected` where `isConnected[i][j] = 1` if the `ith` city and the `jth` city are directly connected, and `isConnected[i][j] = 0` otherwise.

Return _the total number of **provinces**_.

**Example 1:**

![](https://assets.leetcode.com/uploads/2020/12/24/graph1.jpg)

**Input:** isConnected = `[[1,1,0],[1,1,0],[0,0,1]]`
**Output:** 2

**Example 2:**

![](https://assets.leetcode.com/uploads/2020/12/24/graph2.jpg)

**Input:** isConnected = `[[1,0,0],[0,1,0],[0,0,1]]`
**Output:** 3

**Constraints:**

- `1 <= n <= 200`
- `n == isConnected.length`
- `n == isConnected[i].length`
- `isConnected[i][j]` is `1` or `0`.
- `isConnected[i][i] == 1`
- `isConnected[i][j] == isConnected[j][i]`

# Using BFS 
TC - > O(n²)
```java
class Solution {
    private void bfs(int[][] adj, int vis[], int node) {
        vis[node] = 1;
        Queue<Integer> queue = new LinkedList<>();
        queue.offer(node);
        while (!queue.isEmpty()) {
            int vertex = queue.poll();
            for (int j = 0; j < adj.length; j++) {
                if (adj[vertex][j] == 1 && vis[j] == 0) {
                    vis[j] = 1;
                    queue.offer(j);
                }
            }
        }
    }
    public int findCircleNum(int[][] isConnected) {
        int n = isConnected.length;
        int vis[] = new int[n];
        int components = 0;
        for (int node = 0; node < n; node++) {
            if(vis[node] == 0) {
                components++;
                bfs(isConnected, vis, node);
            }
        }
        return components;
    }
}
```

# Using DFS

TC - > O(n²)
```java
class Solution {
    private void dfs(int[][] adj, int vis[], int node) {
        vis[node] = 1;
        for (int j = 0; j < adj.length; j++) {
            if(adj[node][j] == 1 && vis[j] == 0) {
                dfs(adj, vis, j);
            }
        }
    }
    public int findCircleNum(int[][] isConnected) {
        int n = isConnected.length;
        int vis[] = new int[n];
        int components = 0;
        for (int node = 0; node < n; node++) {
            if(vis[node] == 0) {
                components++;
                dfs(isConnected, vis, node);
            }
        }
        return components;
    }
}
```

```java
class Solution {
    private void dfs(int[][] adj, boolean vis[], int node) {
        vis[node] = true;
        for (int j = 0; j < adj.length; j++) {
            if(adj[node][j] == 1 && !vis[j]) {
                dfs(adj, vis, j);
            }
        }
    }
    public int findCircleNum(int[][] isConnected) {
        int n = isConnected.length;
        boolean vis[] = new boolean[n];

        int components = 0;
        for (int node = 0; node < n; node++) {
            if(!vis[node]) {
                components++;
                dfs(isConnected, vis, node);
            }
        }
        return components;
    }
}
```
