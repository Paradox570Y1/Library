[743. Network Delay Time](https://leetcode.com/problems/network-delay-time/)

You are given a network of `n` nodes, labeled from `1` to `n`. You are also given `times`, a list of travel times as directed edges `times[i] = (ui, vi, wi)`, where `ui` is the source node, `vi` is the target node, and `wi` is the time it takes for a signal to travel from source to target.

We will send a signal from a given node `k`. Return _the **minimum** time it takes for all the_ `n` _nodes to receive the signal_. If it is impossible for all the `n` nodes to receive the signal, return `-1`.

**Example 1:**

![](https://assets.leetcode.com/uploads/2019/05/23/931_example_1.png)

**Input:** times = `[[2,1,1],[2,3,1],[3,4,1]]`, n = 4, k = 2
**Output:** 2

**Example 2:**

**Input:** times = `[[1,2,1]]`, n = 2, k = 1
**Output:** 1

**Example 3:**

**Input:** times = `[[1,2,1]]`, n = 2, k = 2
**Output:** -1

**Constraints:**

- `1 <= k <= n <= 100`
- `1 <= times.length <= 6000`
- `times[i].length == 3`
- `1 <= ui, vi <= n`
- `ui != vi`
- `0 <= wi <= 100`
- All the pairs `(ui, vi)` are **unique**. (i.e., no multiple edges.)


# Using Dijkstra Algorithm

```java
class Solution {
    public int networkDelayTime(int[][] times, int n, int k) {
        int minTime[] = new int[n+1];
        Arrays.fill(minTime, (int) 1e9);

        List<List<int[]>> adjList = new ArrayList<>();

        for (int i = 0; i <= n; i++) {
            adjList.add(new ArrayList<>());
        }
        for (int time[] : times) {
            int u = time[0];
            int v = time[1];
            int w = time[2];
            adjList.get(u).add(new int[]{v, w});
        }

        PriorityQueue<int[]> pq = new PriorityQueue<>((a, b) -> a[1] - b[1]);
        pq.offer(new int[]{k, 0});
        minTime[k] = 0;
        while (!pq.isEmpty()) {
            int[] node = pq.poll();
            int u = node[0];
            int du = node[1];
            for (int[] next : adjList.get(u)) {
                int v = next[0];
                int dv = next[1];
                if (du + dv < minTime[v]) {
                    minTime[v] = du + dv;
                    pq.offer(new int[]{v, minTime[v]});
                }
            }
        }
        int maxTimeNeeded = minTime[1];
        for (int i = 2; i < n+1; i++) {
            maxTimeNeeded = Math.max(minTime[i], maxTimeNeeded);
        }
        if (maxTimeNeeded == (int) 1e9) return -1;
        return maxTimeNeeded;
    }
}
```

```java
class Solution {
    public int networkDelayTime(int[][] times, int n, int k) {
        int minTime[] = new int[n+1];
        Arrays.fill(minTime, (int) 1e9);

        int[][] adjMatrix = new int[n+1][n+1];
        for (int i = 0; i <= n; i++) {
            Arrays.fill(adjMatrix[i], (int) 1e9);
        }
        for (int time[] : times) {
            int u = time[0];
            int v = time[1];
            int w = time[2];
            adjMatrix[u][v] = w;
        }

        PriorityQueue<int[]> pq = new PriorityQueue<>((a, b) -> a[1] - b[1]);
        pq.offer(new int[]{k, 0});
        minTime[k] = 0;
        while (!pq.isEmpty()) {
            int[] node = pq.poll();
            int u = node[0];
            int du = node[1];
            if (du > minTime[u]) continue; // This is pruning
            // It avoids relaxing edges from an outdated / worse state.
            for (int v = 1; v <= n; v++) {
                if (adjMatrix[u][v] != (int) 1e9) {
                    int dv = adjMatrix[u][v];
                    if (du + dv < minTime[v]) {
                        minTime[v] = du + dv;
                        pq.offer(new int[]{v, minTime[v]});
                    }
                }
            }
        }
        int maxTimeNeeded = minTime[1];
        for (int i = 2; i < n+1; i++) {
            maxTimeNeeded = Math.max(minTime[i], maxTimeNeeded);
        }
        if (maxTimeNeeded == (int) 1e9) return -1;
        return maxTimeNeeded;
    }
}
```

# Why returning early fails here?
- This is because the question is not how much time it takes to reach nth node but how much time it takes to reach all n nodes in minimum time hence all nodes should be processed first.


# Using DFS + Pruning (Not Recommended for such Problems)
- Can not use adjacency matrix here because in that i cannot sort neighbors of a node for early pruning (so that i can access nodes which require less time first).

Results in Stack Overflow

```java
            Collections.sort(adjList.get(i), (a, b) -> b[1] - a[1]);

        dfs(k, 0);

        int res = -1;
        for (int i = 1; i <= n; i++) {
            res = Math.max(res, dist[i]);

        return res;

    private void dfs(int u, int time) {
        if (dist[u] < time) return;
        dist[u] = time;
        for (int[] next : adjList.get(u)) {
            int v = next[0];
            int w = next[1];
            dfs(v, time + w);
        }
    }
}
```

- Instead of recursive DFS, use your own stack (or queue) so Java doesn’t blow the call stack.

Iterative implementation of Stack Runs Barely 
- Why sort in descending order? (lets analyze)
	- Without any sort my code took `906ms `to run
	- If i sorted it in ascending order then it took `2025ms` to run
	- When i sorted it in descending then it took `56ms` to run
	- So this happens bcz when i sort in descending order , i get neighbors with higher weights first , then i put them in stack first, over which neighbors with lower weights are placed. Due to this i process lower weights first which skips many processing for higher weights.
```java
class Solution {
    int dist[];
    List<List<int[]>> adjList;
    int n;
    public int networkDelayTime(int[][] times, int n, int k) {
        this.n = n;
        this.dist = new int[n+1];
        Arrays.fill(dist, (int) 1e9);
        this.adjList = new ArrayList<>();
        for (int i = 0; i <= n; i++) {
            adjList.add(new ArrayList<>());
        }

        for (int time[] : times) {
            int u = time[0];
            int v = time[1];
            int w = time[2];
            adjList.get(u).add(new int[]{v, w});
        }

        for (int i = 1; i <= n; i++) {
            Collections.sort(adjList.get(i), (a, b) -> b[1] - a[1]);
        }

        Deque<int[]> stack = new ArrayDeque<>();
        stack.push(new int[]{k, 0});

        while (!stack.isEmpty()) {
            int node[] = stack.pop();
            int u = node[0];
            int du = node[1];
            if (dist[u] <= du) continue;
            dist[u] = du;
            for (int[] next : adjList.get(u)) {
                int v = next[0];
                int w = next[1];
                stack.push(new int[]{v, du + w});
            }
        }
        int res = -1;
        for (int i = 1; i <= n; i++) {
            res = Math.max(res, dist[i]);
        }
        if (res == (int) 1e9) return -1;
        return res;
    }
}
```
