[424. Longest Repeating Character Replacement](https://leetcode.com/problems/longest-repeating-character-replacement/)


You are given a string `s` and an integer `k`. You can choose any character of the string and change it to any other uppercase English character. You can perform this operation at most `k` times.

Return _the length of the longest substring containing the same letter you can get after performing the above operations_.

**Example 1:**

**Input:** s = "ABAB", k = 2
**Output:** 4
**Explanation:** Replace the two 'A's with two 'B's or vice versa.

**Example 2:**

**Input:** s = "AABABBA", k = 1
**Output:** 4
**Explanation:** Replace the one 'A' in the middle with 'B' and form "AABBBBA".
The substring "BBBB" has the longest repeating letters, which is 4.
There may exists other ways to achieve this answer too.

**Constraints:**

- `1 <= s.length <= 105`
- `s` consists of only uppercase English letters.
- `0 <= k <= s.length`

```java
class Solution {
    public int characterReplacement(String s, int k) {
        int res = 1;
        for (int i = 0; i < s.length(); i++) {
            StringBuilder sb = new StringBuilder();
            for (int j = i; j < s.length();  j++) {
                sb.append(s.charAt(j));
                Map<Character, Integer> freqMap = new HashMap<>();
                int maxFreq = 1;
                for (int p = 0; p < sb.length(); p++) {
                    int freq = freqMap.merge(sb.charAt(p), 1, Integer::sum);
                    maxFreq = Math.max(maxFreq, freq);
                }
                int wildCards = sb.length() - maxFreq;
                if(wildCards <= k) res = Math.max(res, sb.length());
            }
        }
        return res;
    }
}
```



# Using Sliding Window


```java
class Solution{
  public int characterReplacement(String s, int k) {
    int left = 0;
    Map<Character, Integer> freqMap = new HashMap<>();
    int maxFreq = 1;
    int res = 0;
    for (int right = 0; right < s.length(); right++) {
      int count = freqMap.merge(s.charAt(right), 1, Integer::sum);
      maxFreq = Math.max(maxFreq, count);
      while ((right-left+1) - maxFreq > k) {
        freqMap.merge(s.charAt(left), -1, Integer::sum);
        left++;
      }
      res = Math.max(res, right - left + 1);
    }
    return res;
  }
}
```


OR

- `if` works because once i get the max window for some moment then the max frequency should be more than earlier to have any impact on the result, in between it doesn't matter if the frequency are less or equal to max frequency for any character, unless some character has more frequency  than max , result remains same.
- `when` works because when `k` gets exhausted and a new character comes (which is not the character with max frequency), then if the back of window has the most frequent character in continuity , then it removes all of them because it doesn't matter if that character comes again after current new character, the chain of substring with max frequency is already broken due to a wildcard it can't continue further but yes with same character a new series can begun but will never be connected to the removed one.
- Primary factor that i discovered due to which `while` and `if` both works fine:
	- That is the fact that it should be substring , so the wildcards break the chain and you need to start a new one.
		- In case of while you immediately start a new one, it ensures progress with every right move.
		- In case of if you carry on window to remain same size as current answer which keeps the window invalid and next few right expansions are wasted.

# More optimizations (not required for interview) 
// `while` is giving me better runtime
```java
class Solution{
  public int characterReplacement(String s, int k) {
    int left = 0;
    int hash[] = new int[26];
    int maxFreq = 1;
    int res = 0;
    for (int right = 0; right < s.length(); right++) {
      int ch = s.charAt(right) - 'A';
      hash[ch]++;
      maxFreq = Math.max(maxFreq,hash[ch]);
      while ((right-left+1) - maxFreq > k) {
        ch = s.charAt(left++) - 'A';
        hash[ch]--;
      }
      res = Math.max(res, right-left+1);
    }
    return res;
  }
}
```
