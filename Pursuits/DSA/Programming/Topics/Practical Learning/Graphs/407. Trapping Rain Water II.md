[407. Trapping Rain Water II](https://leetcode.com/problems/trapping-rain-water-ii/)
Given an `m x n` integer matrix `heightMap` representing the height of each unit cell in a 2D elevation map, return _the volume of water it can trap after raining_.

**Example 1:**

![|200](https://assets.leetcode.com/uploads/2021/04/08/trap1-3d.jpg)

**Input:** heightMap = `[[1,4,3,1,3,2],[3,2,1,3,2,4],[2,3,3,2,3,1]]`
**Output:** 4
**Explanation:** After the rain, water is trapped between the blocks.
We have two small ponds 1 and 3 units trapped.
The total volume of water trapped is 4.

**Example 2:**

![](https://assets.leetcode.com/uploads/2021/04/08/trap2-3d.jpg)

**Input:** heightMap = `[[3,3,3,3,3],[3,2,2,2,3],[3,2,1,2,3],[3,2,2,2,3],[3,3,3,3,3]]`
**Output:** 10

**Constraints:**

- `m == heightMap.length`
- `n == heightMap[i].length`
- `1 <= m, n <= 200`
- `0 <= heightMap[i][j] <= 2 * 104`


# Using BFS
- Filling water inwards from minimum level it can reach based on boundary.

```java
class Solution {
    public int trapRainWater(int[][] heightMap) {
        int n = heightMap.length, m = heightMap[0].length;
        PriorityQueue<int[]> pq = new PriorityQueue<>((a, b) -> a[2] - b[2]);
        boolean vis[][] = new boolean[n][m];
        for (int i = 0; i < n; i++) {
            pq.offer(new int[]{i, 0, heightMap[i][0]});
            vis[i][0] = true;
            if(m-1 != 0) {
                pq.offer(new int[] {i, m-1, heightMap[i][m-1]});
                vis[i][m-1] = true;
            }
        }
        for (int j = 1; j < m-1; j++) {
            pq.offer(new int[]{0, j, heightMap[0][j]});
            vis[0][j] = true;
            if(n-1 != 0) {
                pq.offer(new int[]{n-1, j, heightMap[n-1][j]});
                vis[n-1][j] = true;
            }
        }

        int res = 0;
        int dirs[][] = {{-1, 0}, {0, -1}, {1, 0}, {0, 1}};
        while (!pq.isEmpty()) {
            int[] parent = pq.poll();
            for (int[] dir : dirs) {
                int x = dir[0] + parent[0];
                int y = dir[1] + parent[1];
                if (x>=0 && y>=0 && x<n && y<m && !vis[x][y]) {
                    if (heightMap[x][y] < parent[2]) {
                        res += parent[2] - heightMap[x][y];
                    }
                    vis[x][y] = true;
                    pq.offer(new int[] {x, y, Math.max(heightMap[x][y], parent[2])});
                }
            }
        }
        return res;
    }
}
```
