[64. Minimum Path Sum](https://leetcode.com/problems/minimum-path-sum/)

Given a `m x n` `grid` filled with non-negative numbers, find a path from top left to bottom right, which minimizes the sum of all numbers along its path.

**Note:** You can only move either down or right at any point in time.

**Example 1:**

![](https://assets.leetcode.com/uploads/2020/11/05/minpath.jpg)

**Input:** grid = `[[1,3,1],[1,5,1],[4,2,1]]`
**Output:** 7
**Explanation:** Because the path 1 → 3 → 1 → 1 → 1 minimizes the sum.

**Example 2:**

**Input:** grid = `[[1,2,3],[4,5,6]]`
**Output:** 12

**Constraints:**

- `m == grid.length`
- `n == grid[i].length`
- `1 <= m, n <= 200`
- `0 <= grid[i][j] <= 200`


# Using recursion

```java
class Solution {
    private int ans = Integer.MAX_VALUE;
    private void helper(int[][] grid,int row,int col,int sum){
        if(row<0 || col<0)return;
        if(row==0 && col==0){
            ans = Math.min(sum+grid[0][0],ans);
            return;
        }
        helper(grid,row-1,col,sum+grid[row][col]);
        helper(grid,row,col-1,sum+grid[row][col]);
    }
    public int minPathSum(int[][] grid) {
        int row = grid.length;
        int col = grid[0].length;
        helper(grid,row-1,col-1,0);
        return ans;
    }
}
```

# Using Memoization

### Wrong way

Gives TLE bcz memoization is not actually working ,you are just going to memoize last index.
You can't assign the final returned value to memo as sum is something which is calculated while reaching the end point, so memo of i and j is going to store whatever sum was calculated during the first path itself and then it will return wrong value on again passing through that path
```java
class Solution
{
    int[][] grid;
    int[][] memo;
    private int helper(int i,int j,int sum)
    {
        if(i==0 && j==0)return memo[i][j] = sum+grid[i][j];
        if(i<0 || j<0)return Integer.MAX_VALUE;
        if(memo[i][j]!=-1)return memo[i][j];
        int left = helper(i,j-1,sum+grid[i][j]);
        int top = helper(i-1,j,sum+grid[i][j]);
        return Math.min(left,top);
    }
    public int minPathSum(int[][] grid)
    {
        int n = grid.length;
        int m = grid[0].length;
        this.grid = grid;
        memo = new int[n][m];
        for(int i=0;i<n;i++){
            for(int j=0;j<m;j++){
                memo[i][j] = -1;
            }
        }
        return helper(n-1,m-1,0);
    }
}
```

# Using Tabulation

```java
class Solution {
    public int minPathSum(int[][] grid) {
        int n = grid.length;
        int m = grid[0].length;
        int dp[][] = new int[n][m];
        for(int i=0;i<n;i++){
            for(int j=0;j<m;j++){
                dp[i][j] = grid[i][j];
                if(j>0 && i==0)dp[i][j]+=dp[i][j-1];
                if(i>0 && j==0)dp[i][j]+=dp[i-1][j];
                if(i>0 && j>0){
                    dp[i][j] += Math.min(dp[i-1][j],dp[i][j-1]);
                }
            }
        }
        return dp[n-1][m-1];
    }
}

static int minSumPath(int n, int m, int[][] matrix) {
        int prev[] = new int[m];

        for (int i = 0; i < n; i++) {
            int temp[] = new int[m];
            for (int j = 0; j < m; j++) {
                if (i == 0 && j == 0)
                    temp[j] = matrix[i][j];
                else {
                    int up = matrix[i][j];
                    if (i > 0)
                        up += prev[j]; 
                    else
                        up += (int) Math.pow(10, 9); 
                    int left = matrix[i][j];
                    if (j > 0)
                        left += temp[j - 1];
                    else
                        left += (int) Math.pow(10, 9);
                    temp[j] = Math.min(up, left);
                }
            }
            prev = temp;
        }
        return prev[m - 1];
    }

        dp[0] = grid[0][0];
        for(int i=0;i<n;i++){

                if(i-1<0 && j-1<0)continue;
                int left = Integer.MAX_VALUE,top = Integer.MAX_VALUE;
                if(j-1>=0)left = dp[j-1];
                if(i-1>=0)top = dp[j];
                dp[j] = grid[i][j] + Math.min(left,top);
            }
        }
        return dp[m-1];
    }
}

class Solution {
    public int minPathSum(int[][] grid) {
        int r = grid.length;
        int c = grid[0].length;
        int dp[] = new int[c];
        Arrays.fill(dp, (int)1e9);
        dp[0] = 0;
        for (int i = 0; i < r; i++) {
            dp[0] += grid[i][0];
            for (int j = 1; j < c; j++) {
                dp[j] = Math.min(dp[j-1], dp[j]) + grid[i][j];
            }
        }
        return dp[c-1];
    }
}
```
out of bound condition is removed by dp itself from the top and from left it is handled by starting the j by 1 as there will always be 1 way to the 0th index
