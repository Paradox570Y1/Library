[1312. Minimum Insertion Steps to Make a String Palindrome](https://leetcode.com/problems/minimum-insertion-steps-to-make-a-string-palindrome/)

Given a string `s`. In one step you can insert any character at any index of the string.

Return _the minimum number of steps_ to make `s` palindrome.

A **Palindrome String** is one that reads the same backward as well as forward.

**Example 1:**

**Input:** s = "zzazz"
**Output:** 0
**Explanation:** The string "zzazz" is already palindrome we do not need any insertions.

**Example 2:**

**Input:** s = "mbadm"
**Output:** 2
**Explanation:** String can be "mbdadbm" or "mdbabdm".

**Example 3:**

**Input:** s = "leetcode"
**Output:** 5
**Explanation:** Inserting 5 characters the string becomes "leetcodocteel".

**Constraints:**

- `1 <= s.length <= 500`
- `s` consists of lowercase English letters.


>Hint : Another way to think about this problem is to count how many deletions are required to make the string palindrome.
>**Approach 1** `Using LCS`,In this approach you try to find the unaffected component(or try to keep longest palindromic portion intact) which is LCS itself while to calculate no. of insertions required or deletions required you can just subtract the LCS count from length of String
>**Approach 2** `Intitutive way`, In this approach use two pointers recursively and in case character are not same then try deleting character from either side by creating sub problems and if same just bring the two pointers 2 steps  closer.
# Recursion

```java
class Solution {
    private String rev(String s){
        StringBuilder sb = new StringBuilder();
        for(int i=s.length()-1;i>=0;i--){
            sb.append(s.charAt(i));
        }
        return sb.toString();
    }
    private int helper(String a,String b,int i,int j){
        if(i==0 || j==0)return 0;
        int cnt = 0;
        if(a.charAt(i-1) == b.charAt(j-1)){
            cnt = 1 + helper(a,b,i-1,j-1);
        }
        else{
            cnt = Math.max(helper(a,b,i-1,j),helper(a,b,i,j-1));
        }
        return cnt;
    }
    public int minInsertions(String s) {
        int n = s.length();
        return n - helper(s,rev(s),n,n);
    }
}
```

# Using Memoization

```java
class Solution {
    int memo[][];
    private String rev(String s){
        StringBuilder sb = new StringBuilder();
        for(int i=s.length()-1;i>=0;i--){
            sb.append(s.charAt(i));
        }
        return sb.toString();
    }
    private int helper(String a,String b,int i,int j){
        if(i==0 || j==0)return 0;
        if(memo[i][j] != -1)return memo[i][j];
        int cnt = 0;
        if(a.charAt(i-1) == b.charAt(j-1)){
            cnt = 1 + helper(a,b,i-1,j-1);
        }
        else{
            cnt = Math.max(helper(a,b,i-1,j),helper(a,b,i,j-1));
        }
        return memo[i][j] = cnt;
    }
    public int minInsertions(String s) {
        int n = s.length();
        memo = new int[n+1][n+1];
        for(int i=0;i<=n;i++){
            Arrays.fill(memo[i],-1);
        }
        return n - helper(s,rev(s),n,n);
    }
}
```

# Using Tabulation

```java
class Solution {
    public int minInsertions(String s) {
        int n = s.length();
        int dp[] = new int[n+2];
        for(int i=1;i<=n;i++){
            int cur[] = new int[n+2];
            for(int j=n;j>=1;j--){
                if(s.charAt(i-1) == s.charAt(j-1)){
                    cur[j] = 1 + dp[j+1];
                }
                else{
                    cur[j] = Math.max(cur[j+1],dp[j]);
                }
            }
            dp = cur;
        }
        return n-dp[1];
    }
}
```

```java
class Solution {
    public int minInsertions(String s) {
        int n = s.length();
        int[] dp = new int[n+1];
        for (int i = 1; i <= n; i++) {
            int prev = 0;
            for (int j = 1;j <= n; j++) {
                int b = dp[j];
                if (s.charAt(i-1) == s.charAt(n-j)) {
                    dp[j] = 1 + prev;
                } else {
                    dp[j] = Math.max(dp[j-1], dp[j]);
                }
                prev = b;
            }
        }
        return n - dp[n];
    }
}
```
