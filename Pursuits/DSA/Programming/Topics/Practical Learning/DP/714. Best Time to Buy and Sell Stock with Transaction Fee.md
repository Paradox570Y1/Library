[714. Best Time to Buy and Sell Stock with Transaction Fee](https://leetcode.com/problems/best-time-to-buy-and-sell-stock-with-transaction-fee/)


You are given an array `prices` where `prices[i]` is the price of a given stock on the `ith` day, and an integer `fee` representing a transaction fee.

Find the maximum profit you can achieve. You may complete as many transactions as you like, but you need to pay the transaction fee for each transaction.

**Note:**

- You may not engage in multiple transactions simultaneously (i.e., you must sell the stock before you buy again).
- The transaction fee is only charged once for each stock purchase and sale.

**Example 1:**

**Input:** prices = [1,3,2,8,4,9], fee = 2
**Output:** 8
**Explanation:** The maximum profit can be achieved by:
- Buying at prices[0] = 1
- Selling at prices[3] = 8
- Buying at prices[4] = 4
- Selling at prices[5] = 9
The total profit is ((8 - 1) - 2) + ((9 - 4) - 2) = 8.

**Example 2:**

**Input:** prices = [1,3,7,5,10,3], fee = 3
**Output:** 6

**Constraints:**

- `1 <= prices.length <= 5 * 104`
- `1 <= prices[i] < 5 * 104`
- `0 <= fee < 5 * 104`




# Recursion

```java
class Solution {
    private int helper(int[] prices,int i,int buy,int fee){
        if(i==prices.length)return 0;
        int profit = 0;
        if(buy == 1){
            int b = -prices[i] + helper(prices,i+1,0,fee);
            int nb = helper(prices,i+1,1,fee);
            profit = Math.max(b,nb);
        } else {
            int s = prices[i] - fee + helper(prices,i+1,1,fee);
            int ns = helper(prices,i+1,0,fee);
            profit = Math.max(s,ns);
        }
        return profit;
    }
    public int maxProfit(int[] prices, int fee) {
        return helper(prices,0,1,fee);
    }
}
```

# Memoization

```java
class Solution {
    int memo[][];
    private int helper(int[] prices,int i,int buy,int fee){
        if(i==prices.length)return 0;
        if(memo[i][buy] != -1)return memo[i][buy];
        int profit = 0;
        if(buy == 1){
            int b = -prices[i] + helper(prices,i+1,0,fee);
            int nb = helper(prices,i+1,1,fee);
            profit = Math.max(b,nb);
        } else {
            int s = prices[i] - fee + helper(prices,i+1,1,fee);
            int ns = helper(prices,i+1,0,fee);
            profit = Math.max(s,ns);
        }
        return memo[i][buy] = profit;
    }
    public int maxProfit(int[] prices, int fee) {
        int n = prices.length;
        memo = new int[n][2];
        for(int i=0;i<n;i++){
            memo[i][0] = memo[i][1] = -1;
        }
        return helper(prices,0,1,fee);
    }
}
```



# DP

```java
class Solution {
    public int maxProfit(int[] prices, int fee) {
        int n = prices.length;
        int dp[][] = new int[n+1][2];
        for(int i=n-1;i>=0;i--){
            dp[i][1] = Math.max(prices[i]- fee + dp[i+1][0],dp[i+1][1]);
            dp[i][0] = Math.max(-prices[i] + dp[i+1][1],dp[i+1][0]);
        }
        return dp[0][0];
    }
}
```


#### Optimizing Space

```java
class Solution {
    public int maxProfit(int[] prices, int fee) {
        int n = prices.length;
        int prev[] = new int[2];
        for(int i=n-1;i>=0;i--){
            int cur[] = new int[2];
            cur[1] = Math.max(prices[i]- fee + prev[0],prev[1]);
            cur[0] = Math.max(-prices[i] + prev[1],prev[0]);
            prev = cur;
        }
        return prev[0];
    }
}
```

```java
class Solution {

        int n = prices.length;
        int prevBuy,prevSell,curBuy,curSell;
        prevBuy = prevSell = 0;
        for(int i=n-1;i>=0;i--){
            curSell = Math.max(prices[i]- fee + prevBuy,prevSell);
            curBuy = Math.max(-prices[i] + prevSell,prevBuy);
            prevBuy = curBuy;
            prevSell = curSell;
        }
        return prevBuy;
    }
}
```

```java
class Solution {
    public int maxProfit(int[] prices, int fee) {
        int pb = (int)-1e9, ps = 0;
        for (int price : prices) {
            int b = Math.max(pb, ps - price);
            int s = Math.max(ps, pb + price - fee);
            pb = b;
            ps = s;
        }
        return ps;
    }
}
```
