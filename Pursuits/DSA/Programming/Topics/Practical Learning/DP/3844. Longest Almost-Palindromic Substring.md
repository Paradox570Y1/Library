[3844. Longest Almost-Palindromic Substring](https://leetcode.com/problems/longest-almost-palindromic-substring/)

You are given a string `s` consisting of lowercase English letters.

A substring is **almost-palindromic** if it becomes a palindrome after removing **exactly** one character from it.

Return an integer denoting the length of the **longest** **almost-palindromic** substring in `s`.

**Example 1:**

**Input:** s = "abca"

**Output:** 4

**Explanation:**

Choose the substring `"**abca**"`.

- Remove `"ab**c**a"`.
- The string becomes `"aba"`, which is a palindrome.
- Therefore, `"abca"` is almost-palindromic.

**Example 2:**

**Input:** s = "abba"

**Output:** 4

**Explanation:**

Choose the substring `"**abba**"`.

- Remove `"a**b**ba"`.
- The string becomes `"aba"`, which is a palindrome.
- Therefore, `"abba"` is almost-palindromic.

**Example 3:**

**Input:** s = "zzabba"

**Output:** 5

**Explanation:**

Choose the substring `"z**zabba**"`.

- Remove `"**z**abba"`.
- The string becomes `"abba"`, which is a palindrome.
- Therefore, `"zabba"` is almost-palindromic.

**Constraints:**

- `2 <= s.length <= 2500`
- `s` consists of only lowercase English letters.


# Using DP
- We are using LPS with partition DP to find longest substring for each string but time complexity of this approach is not that good.
```java
class Solution {
    private int lps(String s, int t) {
        int n = s.length();
        boolean[][] isPal = new boolean[n][n];
        int len = 1;
        boolean isNear = true;
        boolean anyPal = false;
        boolean flag = false;
        for (int i = 0; i < n; i++) {
            isPal[i][i] = true;
            if (i+1 < n && s.charAt(i) == s.charAt(i+1)) {
                isPal[i][i+1] = true;
                anyPal = true;
                if (t >= i-1 && t <= i+2) flag = true;
                len = 2;
            }
        }
        if (anyPal) isNear = flag;
        
        for (int grp = 3; grp <= n; grp++) {
            flag = false;
            anyPal = false;
            for (int i = 0; i <  n-grp+1; i++) {
                int j = i+grp-1;
                if (isPal[i+1][j-1] && s.charAt(i) == s.charAt(j)) {
                    isPal[i][j] = true;
                    anyPal = true;
                    if (t >= i-1 && t <= j+1) flag = true;
                    len = grp;
                }
            }
            if (anyPal) {
                isNear = flag;
            }
        }
        
        return len + ((isNear)?1:0);
    }
    public int almostPalindromic(String s) {
        int res = lps(s, -3);
        for (int i = 0; i < s.length(); i++) {
            int val = lps(s.substring(0, i) + s.substring(i+1), i);
            
            res = Math.max(res, val);
        }
        return res;
    }
}
```
