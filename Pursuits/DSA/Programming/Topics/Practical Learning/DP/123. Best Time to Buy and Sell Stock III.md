[123. Best Time to Buy and Sell Stock III](https://leetcode.com/problems/best-time-to-buy-and-sell-stock-iii/)


You are given an array `prices` where `prices[i]` is the price of a given stock on the `ith` day.

Find the maximum profit you can achieve. You may complete **at most two transactions**.

**Note:** You may not engage in multiple transactions simultaneously (i.e., you must sell the stock before you buy again).

**Example 1:**

**Input:** prices = [3,3,5,0,0,3,1,4]
**Output:** 6
**Explanation:** Buy on day 4 (price = 0) and sell on day 6 (price = 3), profit = 3-0 = 3.
Then buy on day 7 (price = 1) and sell on day 8 (price = 4), profit = 4-1 = 3.

**Example 2:**

**Input:** prices = [1,2,3,4,5]
**Output:** 4
**Explanation:** Buy on day 1 (price = 1) and sell on day 5 (price = 5), profit = 5-1 = 4.
Note that you cannot buy on day 1, buy on day 2 and sell them later, as you are engaging multiple transactions at the same time. You must sell before buying again.

**Example 3:**

**Input:** prices = [7,6,4,3,1]
**Output:** 0
**Explanation:** In this case, no transaction is done, i.e. max profit = 0.

**Constraints:**

- `1 <= prices.length <= 105`
- `0 <= prices[i] <= 105`

# Iterative Brute Way
TC -> O(N²)
```java
class Solution {
    public int maxProfit(int[] prices) {
        int profit = 0;
        int n = prices.length;
        int buy = prices[0];
        int p1 = 0;
        for (int i = 1; i < n; i++) {
            if(prices[i] - buy > 0) p1 = Math.max(prices[i] - buy, p1);
            int buy2 = prices[i];
            int p2 = 0;
            for (int j = i+1; j < n; j++) {
                if (prices[j] - buy2 > 0) p2 = Math.max(prices[j] - buy2, p2);
                buy2 = Math.min(prices[j], buy2);
            }
            buy = Math.min(buy,prices[i]);
            profit = Math.max(profit, p1 + p2);
        }
        return profit;
    }
}
```

# DP
In this we reverse engineer so we sell then buy and again sell and then buy

class Solution {
    public int maxProfit(int[] prices) {
        int n = prices.length;
        int dp[][] = new int[n+1][5];
        int profit = 0;
        for(int i=n-1;i>=0;i--){
            for(int canBuy=3;canBuy>=0;canBuy--){
                if(canBuy%2==0){
                    int buy = -prices[i] + dp[i+1][canBuy+1];
                    int notBuy = dp[i+1][canBuy];
                    profit = Math.max(buy,notBuy);
                }
                else{
                    int sell = prices[i] + dp[i+1][canBuy+1];
                    int notSell = dp[i+1][canBuy];
                    profit = Math.max(sell,notSell);
                } 
                dp[i][canBuy] = profit;
            }
        }
        return dp[0][0];
    }
}

# Using Greedy Inspired 2 pass approach


```java
class Solution {
    public int maxProfit(int[] prices) {
        int n = prices.length;
        int left[] = new int[n];
        int right[] = new int[n];
        int minBuy=prices[0];
        int maxProfit = 0;
        for(int i=1;i<n;i++){
            minBuy = Math.min(minBuy,prices[i]);
            left[i] = Math.max(left[i-1],prices[i]-minBuy);
        }
        int maxSell = prices[n-1];
        maxProfit=0;
        for(int i=n-2;i>=0;i--){
            maxSell = Math.max(maxSell,prices[i]);
            right[i] = Math.max(right[i+1],maxSell-prices[i]);
        }
        maxProfit=0;
        for(int i=0;i<n;i++){
            maxProfit = Math.max(maxProfit,left[i]+right[i]);
        }
        return maxProfit;
    }
}
```


# Greedy way to keep isolation while calculation 
Using Greedy
It's not just greedy per transaction — it's greedy with **profit reinvestment**, which is the smart trick here.
## ✅ Time & Space

- **Time:** O(n)
- **Space:** O(1)
```java
class Solution {
    public int maxProfit(int[] prices) {
        if (prices.length == 0) return 0;

        int buy1 = Integer.MAX_VALUE;
        int profit1 = 0;

        int buy2 = Integer.MAX_VALUE;
        int profit2 = 0;

        for (int price : prices) {
            // The lowest price to buy first stock
            buy1 = Math.min(buy1, price);
            // Maximum profit after selling first stock
            profit1 = Math.max(profit1, price - buy1);

            // The effective lowest price to buy second stock (after first profit)
            buy2 = Math.min(buy2, price - profit1);
            // Maximum profit after selling second stock
            profit2 = Math.max(profit2, price - buy2);
        }

        return profit2;
    }
}
```
