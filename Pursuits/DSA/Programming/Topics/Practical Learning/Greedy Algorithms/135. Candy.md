[135. Candy](https://leetcode.com/problems/candy/)

There are `n` children standing in a line. Each child is assigned a rating value given in the integer array `ratings`.

You are giving candies to these children subjected to the following requirements:

- Each child must have at least one candy.
- Children with a higher rating get more candies than their neighbors.

Return _the minimum number of candies you need to have to distribute the candies to the children_.

**Example 1:**

**Input:** ratings = [1,0,2]
**Output:** 5
**Explanation:** You can allocate to the first, second and third child with 2, 1, 2 candies respectively.

**Example 2:**

**Input:** ratings = [1,2,2]
**Output:** 4
**Explanation:** You can allocate to the first, second and third child with 1, 2, 1 candies respectively.
The third child gets 1 candy because it satisfies the above two conditions.

**Constraints:**

- `n == ratings.length`
- `1 <= n <= 2 * 104`
- `0 <= ratings[i] <= 2 * 104`


# Brute

```java
class Solution {
    class Child{
        int pos;
        int rating;
        Child(int pos,int rating){
            this.pos = pos;
            this.rating = rating;
        }
    }
    public int candy(int[] ratings) {
        int n = ratings.length;
        Child[] child = new Child[n];
        for(int i=0;i<n;i++){
            child[i] = new Child(i,ratings[i]);
        }
        Arrays.sort(child,(a,b)->a.rating-b.rating);
        int hash[] = new int[n];
        for(int i=0;i<n;i++){
            Child cur = child[i];
            int pos = child[i].pos;
            int ranking = child[i].rating;
            hash[pos]=1;
            if(pos>0 && ratings[pos-1]<ratings[pos])hash[pos] = hash[pos-1]+1;
            if(pos<n-1 && ratings[pos+1]<ratings[pos])hash[pos] = Math.max(hash[pos+1]+1,hash[pos]);
        }
        int ans=0;
        for(int candy:hash){
            ans+=candy;
        }
        return ans;
    }
}
```

# Optimal with constant space

```java
class Solution {
    public int candy(int[] ratings) {
        int n= ratings.length;
        int hash[] = new int[n];
        int ans = 1;
        int i=1;
        int candy=1;
        while(i<n){
            if(ratings[i]>ratings[i-1]){
                candy++;
                ans+=candy;
                i++;
            }
            else {
                int prev = candy;
                ans-=candy;
                int j = i;
                while(j<n && ratings[j]<=ratings[j-1])j++;
                int temp = j;
                j--;
                candy=1;
                while(j>=i){
                    ans+=candy;
                    if(ratings[j]<ratings[j-1])candy++;
                    else candy = 1;
                    j--;
                }
                ans += Math.max(prev,candy);
                candy=1;
                i=temp;
            }
        }
        return ans;
    }
}
```


# Using slope Approach
TC-> O(N)
SC-> O(1)
```java
class Solution {
    public int candy(int[] ratings) {
        int n = ratings.length;
        int ans=1;
        int i=1;
        while(i<n){
            if(ratings[i] == ratings[i-1]){
                ans++;
                i++;
                continue;
            }
            int up = 1;
            while(i<n && ratings[i] > ratings[i-1]){
                up++;
                ans+=up;
                i++;
            }
            int down = 1;
            while(i<n && ratings[i] < ratings[i-1]){
                ans+=down;
                down++;
                i++;
            }
            if(down>up){
                ans+=down-up;
            }
        }   
        return ans;
    }
}
```
