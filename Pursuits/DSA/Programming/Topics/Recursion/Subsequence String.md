## Problem statement

Send feedback

You are given a string 'STR' containing lowercase English letters from a to z inclusive. Your task is to find all non-empty possible subsequences of 'STR'.

```
A Subsequence of a string is the one which is generated by deleting 0 or more letters from the string and keeping the rest of the letters in the same order.
```

Detailed explanation ( Input/output format, Notes, Images )

**Constraints:**

```
1 <= T <= 10 
1 <=  |STR| <= 16

Where |STR| represents the length of the string 'STR'.

Time Limit: 1 sec
```

**Sample Input 1:**

```
1 
abc
```

**Sample Output 1:**

```
a ab abc ac b bc c
```

**Explanation of sample input 1:**

```
All possible subsequences of abc are :  
“a” , “b” , “c” , “ab” , “bc” , “ac”, “abc”
```

**Sample Input 2:**

```
1
bbb
```

**Sample Output 2:**

```
b b b bb bb bb bbb
```

# Brute
Using Bit Manipulation
TC-> O(n x 2^n)
```java
import java.util.* ;
import java.io.*; 
import java.util.ArrayList;

class Solution {
    public static ArrayList<String> subsequences(String str) {
        // Write your code here
        ArrayList<String> ans = new ArrayList<>();
        int n = str.length();
        int limit = 1<<n;
        // start from i=0 if you want to include empty substring
        for(int i=1;i<limit;i++){
            String s="";
            for(int j=0;j<n;j++){
                if((i & (1<<j))!=0)s+=str.charAt(j);
            }
            ans.add(s);
        }
        return ans;
    }
}
```
# Better
Using Recursion
TC-> O(2^n)
```java
import java.util.* ;
import java.io.*; 
import java.util.ArrayList;

class Solution {
    static void generate(String s,String out,ArrayList<String> ans){
        if(s.length() == 0){
            if(!out.isEmpty())
            ans.add(out);
            return;
        }
        generate(s.substring(1),out+s.charAt(0),ans);
        generate(s.substring(1),out,ans);
    }
    public static ArrayList<String> subsequences(String str) {
        // Write your code here
        ArrayList<String> ans = new ArrayList<>();
        generate(str,"",ans);
        return ans;
    }
}
```
