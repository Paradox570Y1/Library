#### What is Time Complexity?
- Rate of increase in time with respect to size of input.
- No. of Iterations with respect to size of input.
![[Pasted image 20240705114153.png|400]]

> Big O -> worst
> Omega -> Best
> Theta -> Avg
>

	- small o { o }
	- small omega { ω }
