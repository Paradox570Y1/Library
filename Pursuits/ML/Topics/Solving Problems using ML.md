#### How machine learning solves Problems?
######    Traditional Problem Solving vs Machine Learning
   `Traditional problem solving: ` In this with a software a person analysis the path and engineers the solution to solve that problem. For many real world examples this takes a lot of time.
   Imagine writing a program which can detect if a cat is present in an image or not. So, traditional problem solving requires attention to details like different types of cats  and various poses a cat might be in.
   
   Therefore, this process can be laborious (or even impossible) because a correct solution would need to take a vast number of edge cases into consideration.
   ![[Pasted image 20240626102636.png|400]]
   
   `Machine Learning: ` In ML the problem solver abstracts away part of their solution as a flexible component called *Model*, and uses a special program called a **model training algorithm** to adjust that model to real-world data. The result is a trained model which can be used to predict outcomes that are not part of the dataset used to train it.
   *In a way ML automates some of the statical reasoning and pattern matching that problem solver will traditionally do. The flexibility of the model is the key here.*

   **The overall goal is to use a model created by a model-training algorithm to generate predictions or find patterns in data that can be used to solve a problem.**
