### 📘 IAM (Identity and Access Management)

IAM is a **framework** used to control:

- **Who** can access what (identity}
- **What** they can do (access/permissions)
- **Where** and **when** they can do it

---

### 🧩 IAM Integration in Cloud (e.g., AWS, Azure, GCP)

In cloud services like AWS:

- **IAM Integration** lets you **connect external identity providers** (like Google, Active Directory, Okta) to AWS IAM.
    
- It allows **Single Sign-On (SSO)**, **[[federated access]]**, and **role-based permissions**.
