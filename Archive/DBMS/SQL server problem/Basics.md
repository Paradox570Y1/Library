#### Data vs Info
- Processed Data is known as Information.
- If the given data is just data or information ,it is all about perspective.



__Q. Describe the case when data will become a information or vice versa?__
- __Case 1:__ Modi is Prime Minister it is information for Modi and other politician but it is data for public.
- __Case 2__: In a match between IND and AUS if IND won then it is information for both AUS and IND (as it decides whether they will qualify for next round) but data for Public.
