#### Difference between End Device and Host
`Host`
- **Definition**: A host is any device that can send and receive data on a network. It typically refers to computers, servers, or any device with an IP address.
- **Functions**: Hosts participate in network communication, run applications, provide services, and handle user tasks. Examples include web servers, email servers, and personal computers.
- **Networking**: In a networking context, hosts have unique IP addresses and can be identified individually within a network.

`End Device`
- **Definition**: An end device is any device that represents the source or destination of data in a network. It is at the "end" of a network path, meaning it does not route traffic to other devices.
- **Functions**: End devices typically refer to devices used by end-users, such as smartphones, tablets, laptops, and IoT devices. They usually interact with the network to access services, send/receive data, or execute tasks initiated by users.
- **Networking**: End devices also have IP addresses and often rely on intermediate network devices (like routers and switches) to communicate with other devices.

# Basic Terminologies

• `Network Interface Card (NIC)` - A NIC physically connects the end device to the network.
• `Physical Port`- A connector or outlet on a networking device where the media connects to an end device or another networking device.
• `Interface` - Specialized ports on a networking device that connect to individual networks. Because routers connect networks, the ports on a router are referred to as network interfaces.
Note: The terms port and interface are often used interchangeably.
