## 🥇 1. Amazon

**Job Roles Offered:** Software Development Engineer (SDE), Data Engineer, QA Engineer  
**Avg Salary Range:** 12–20 LPA for freshers; high growth potential annually  
**Work-Life Balance:** Moderate—high expectations but flexibility & ownership  
**Overview:** Industry-leading global tech culture, customer-centric, innovation-driven, with mentorship and growth through Amazon’s Leadership Principles.  
**Hiring Timeline:** Campus drives often in Aug–Sep; occasional apply-through-pre-placement-offer (PPO) or challenge rounds in Jan–Mar.  
**Application Process:**

1. Apply via Chitkara CAS portal or Amazon early-access links.
    
2. Online assessment: coding + logical reasoning.
    
3. Technical rounds: DS&Algo, System Design, Behavioral (leadership-oriented).
    
4. Offer & OA internship conversion.  
    **Requirements:** Proficiency in C/C++/Java/Python, strong DS/A, problem-solving prowess, code efficiency, collaborative mindset.  
    **Insider Tip:** Senior alumni note preparing via Amazon’s **OA practice questions** and conducting mock interviews (DSA + system design) propelled success.

## 🥈 2. Microsoft

**Job Roles Offered:** SDE, Program Manager, Data Scientist/Engineer  
**Avg Salary Range:** 12–18 LPA; steady increments and stock grants  
**Work-Life Balance:** Strong—focus on innovation with personal time  
**Overview:** Inclusive, growth-oriented culture with impact-driven teams; values a “growth mindset.”  
**Hiring Timeline:** Campus drives in Sep–Oct; internship conversions in Mar–Apr.  
**Application Process:**

1. CAS application → online tests: coding + aptitude.
    
2. 2–3 interviews: DS/A, design, behavioral.
    
3. Offer.  
    **Requirements:** Strong coding skills, analytical approach, Microsoft-specific culture fit ([STAR](obsidian://open?vault=Lessons&file=Interns%2FSTAR%20Method) stories).  
    **Insider Tip:** Use **LeetCode’s Microsoft tag**, and structure answers using the [STAR](obsidian://open?vault=Lessons&file=Interns%2FSTAR%20Method) format for behavioral rounds.
